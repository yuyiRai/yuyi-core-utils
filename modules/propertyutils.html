getPropertyFieldByCreate

  • getPropertyFieldByCreate<V>(main: IKeyValueMap, ...proteryNames: PrototeryMatcher[]): V
    • 定义于 PropertyUtils/getPropertyFieldByCreate.ts:25

    从对象中提取成员，不存在则新建一个成员（默认为一个空对象） 提取对象逐渐深入，一个对象一次只能提取一个成员（或他的子成员的子成员）

    Example

    const obj = {}
const a = getPropertyFieldByCreate(obj, 'a', ['a'], ['a', 'b'])

console.log(a) // 'b'
console.log(obj) // {"a":{"a":{"a":"b"}}}
