Optional autoCompleteType

autoCompleteType: "cc-csc" | "cc-exp" | "cc-exp-month" | "cc-exp-year" | "cc-number" | "email" | "name" | "password" | "postal-code" | "street-address" | "tel" | "username" | "off"

Determines which content to suggest on auto complete, e.g.username. To disable auto complete, use off.

Optional disableFullscreenUI

disableFullscreenUI: boolean
  • 定义于 node_modules/@types/react-native/index.d.ts:1154

When false, if there is a small amount of space available around a text input (e.g. landscape orientation on a phone), the OS may choose to have the user edit the text inside of a full screen text input mode. When true, this feature is disabled and users will always edit the text directly inside of the text input. Defaults to false.
